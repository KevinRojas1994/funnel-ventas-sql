# Funnel de Ventas – Análisis con SQL

Este proyecto fue desarrollado como parte de un proceso de aprendizaje en Análisis de Datos.  
El objetivo es analizar un funnel de ventas utilizando SQL para identificar pérdidas de clientes
y calcular tasas de conversión entre cada etapa.

## Objetivo del proyecto
Analizar el desempeño de un funnel de ventas y detectar en qué etapas se presenta la mayor
pérdida de clientes, con el fin de apoyar la toma de decisiones comerciales.

## Descripción del dataset
El dataset representa un proceso de ventas con las siguientes etapas:
- Lead
- Contactado
- Oportunidad
- Venta cerrada

Cada registro corresponde a un cliente que avanza o se pierde dentro del funnel.

## Herramientas utilizadas
- SQL
- Excel
- GitHub

## Proceso de análisis
1. Revisión y limpieza de los datos.
2. Clasificación de clientes por etapa del funnel.
3. Cálculo de métricas por etapa utilizando SQL.
4. Análisis de tasas de conversión entre etapas.

## Consultas SQL utilizadas
- `SELECT`, `WHERE`
- `GROUP BY`
- Funciones agregadas (`COUNT`)
- Subconsultas para el cálculo de métricas

## Conclusiones
El análisis permitió identificar las etapas del funnel con mayor pérdida de clientes, lo que
representa oportunidades de mejora en el proceso comercial.
